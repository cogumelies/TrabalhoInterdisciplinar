# Trabalho Interdisciplinar

Integrantes: Ana Lara Perini Calzi, Jose Henrique Pereira, Júlio César Dossi Biasutti

## Descrição

O **Trabalho Interdisciplinar** é um projeto de desenvolvimento web criado com o objetivo de aplicar os conhecimentos adquiridos nas disciplinas de programação, desenvolvimento de interfaces e lógica computacional. O sistema simula uma plataforma de gerenciamento de produtos e pedidos, oferecendo uma interface simples, intuitiva e organizada para o usuário.

O projeto foi desenvolvido utilizando tecnologias fundamentais da web — **HTML5**, **CSS3** e **JavaScript** — seguindo o conceito de separação entre estrutura, apresentação e comportamento da aplicação.

## Objetivos

Este projeto tem como finalidade:

* Aplicar conceitos de desenvolvimento web.
* Desenvolver páginas responsivas e organizadas.
* Utilizar JavaScript para adicionar interatividade ao sistema.
* Praticar a organização de arquivos e boas práticas de programação.
* Integrar os conhecimentos adquiridos durante o curso em um único projeto.

## Funcionalidades

O sistema possui diferentes páginas para atender às principais funcionalidades da aplicação:

* Página inicial.
* Catálogo de produtos.
* Cadastro de produtos.
* Realização de pedidos.
* Página de contato.
* Página institucional (Sobre).
* Tela de confirmação das operações realizadas.

## Organização do Código

O projeto foi estruturado de forma a facilitar a manutenção e o entendimento do código:

* **HTML**: responsável pela estrutura das páginas.
* **CSS**: define a identidade visual, o layout e a responsividade.
* **JavaScript**: implementa as funcionalidades e a interação com o usuário.

Essa separação torna o código mais organizado, reutilizável e de fácil manutenção.

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/cogumelies/TrabalhoInterdisciplinar.git
```

2. Acesse a pasta do projeto.

3. Abra o arquivo `land.html` em qualquer navegador moderno.

Não é necessário instalar dependências ou utilizar um servidor, pois o projeto foi desenvolvido utilizando apenas tecnologias front-end.
